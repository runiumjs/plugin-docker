# @runium/plugin-docker

[![npm](https://img.shields.io/npm/v/@runium/plugin-docker)](https://www.npmjs.com/package/@runium/plugin-docker)
[![license](https://img.shields.io/npm/l/@runium/plugin-docker)](LICENSE)

Runium plugin for Docker support — provides tasks for working with Docker and Docker Compose.

## Installation

```bash
npm install @runium/plugin-docker
```

or

```bash
yarn add @runium/plugin-docker
```
